# 404 Page — Design Spec

**Date:** 2026-03-20
**Status:** Approved

## Overview

Add a basic 404 error page to the St. Nicholas Orthodox Church website.

## Implementation

**File:** `src/pages/404.astro`

Astro automatically serves `src/pages/404.astro` for any unmatched route.

### Structure

- `<Layout title="404 — Page Not Found" description="Page not found.">`
- `<div class="content">` wrapper (consistent with all other pages)
- `<PageH1 title="404" subheader="This page could not be found." />`
- Short paragraph: "The page you're looking for doesn't exist or has been moved."
- Link button: `<a href="/" class="button primary medium">Go to Home Page</a>`

### Styles

No new SCSS needed. Uses existing `.content`, `.button`, `.button.primary`, and `.button.medium` classes from the global stylesheet.

## Out of Scope

- Custom illustrations or imagery
- Church-themed messaging
- Animations
